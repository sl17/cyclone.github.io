## Css盒模型


### 盒模型有两种 

- 标准盒模型
- IE盒模型

#### 标准盒模型

```
当设置为box-sizing:content-box时，将采用标准模式解析计算，也是默认模式；
```

> Width = 内容宽度(contnet) + padding + border + margin

![I and My friends]({{site.baseurl}}/assets/img/demo/201908/2019-08-15_084658.png)

#### IE盒模型(低版本)

```
当设置为box-sizing:border-box时，将采用怪异模式解析计算；
```
> Width = 内容宽度(contnet + padding + border ) + margin
