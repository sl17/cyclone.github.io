## 二进制与十进制转化


例如：12转二进制

```
  12/2 = 6 余 0
  6/2 =  3 余 0
  3/2 =  1 余 1
  1/2 =  0 余 1
  
  那么12的二进制就是 1100
  
```

例如：1100转十进制

```
	...16	8	4	2	1   
        1	1	0	0
——————————————
			8	4	0	0
——————————————
			8 + 4 = 12    
	   
```
