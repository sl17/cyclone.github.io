## Flex布局垂直居中

### 方案一：
```
//style
.wrap{
  display: flex;
  justify-content: center;
}
.child{
  align-self: center;
}
//html
<div class="wrap">
  <div class="child"></div>
</div>
```

### 方案二：
```
//style
.wrap{
  display: flex;
  justify-content: center;
  align-items: center;
}
.child{
  background: pink;
  overflow: hidden;
}
//html
<div class="wrap">
  <div class="child"></div>
</div>
```
