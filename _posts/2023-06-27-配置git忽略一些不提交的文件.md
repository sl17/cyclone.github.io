## 配置git忽略一些不提交的文件


#### 配置git忽略一些不提交的文件
在.gitignore 文件中配置要忽略的文件名词和后缀

```
.DS_Store
node_modules
/dist
package-lock.json
package.json
vue.config.js

# local env files
.env.local
.env.*.local

# Log files
npm-debug.log*
yarn-debug.log*
yarn-error.log*

# Editor directories and files
.idea
.vscode
*.suo
*.ntvs*
*.njsproj
*.sln
*.sw*

```


先把本地缓存删除（改变成未track状态），本地不能有未提交的更改

再刷新 .gitignore 文件
```
git commit -m 'update .gitignore'
```
