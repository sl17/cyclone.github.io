### vue项目字典处理


#### vue项目字典处理

`js文件`

```
const op = {
	aa:[
		{label: 'ass',value:'aa1'},
		{label: 'ass1',value:'aa2'},
	],
	bb:[
		{label: 'bsbs',value:'bb1'},
		{label: 'bsbs1',value:'bb2'},
	]
}
export default {
	get (name) {
    return op[name]
  },
  label (name, value) {
    const ret = op[name]
      ? op[name].find(item => {
          return item.value === value
        })
      : null
    return ret ? ret.label : value
  }
}

```

`main.js` 文件导入字典就可以在vue 里用
```
import op from '@/utils/op.js' //字典项
```
