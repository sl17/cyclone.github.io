## A New Post

### 安装

```
npm install less less-loader --save-dev
```

### 配置

>在build文件夹下的webpack.base.conf.js的rules里面添加配置

```
在：
module: {
  rules: [
    {
      其他内容
    },
    *在此添加*
  ]
}
添加如下代码：
{
  test: /\.less$/,
  loaders: 'style-loader!css-loader!less-loader'
}
```

### 页面配置

```
<style lang="less"></style>
```
