### 关于Css的引入字体、轮廓属性、监听横/竖屏幕


自定义字体:font-face

```
@font-face {
 font-family:”刘体";
 src: url("新蒂泡芙体.otf");
}
div{
 font-size: 50px;
 font-family:  “刘体";
}
```

轮廓:不占文本流,只是浮在文本区域周围

```
p{outline: 10px solid red;}
input{ outline: none;}
```

监听横竖屏

```
/*横屏*/
@media all and (orientation:landscape) {
 h1{color: red;}}
/*竖屏*/
@media all and (orientation:portrait){
h1{color: greenyellow;}}
```
