## 关于VS code的一些操作

### 安装中文

- `Ctrl` + `Shift` + `p` 
- 输入 `configure`
- 选择安装其他语言->选择中文

![安装中文]({{site.baseurl}}/assets/img/demo/201909/2019-09-06_00002.png)

### 安装vue高亮

- `Ctrl` + `Shift` + `x`
- 输入 `vetur`
- 安装即可

![vue高亮]({{site.baseurl}}/assets/img/demo/201909/2019-09-06_00003.png)

### 快捷键

- `Ctrl`+`Shift`+`k`	删除整行；
- `Ctrl`+`B`	关闭/打开项目列表；
- `Alt`+`↑`	向上移动整行；
- `Alt`+`↓`	向下移动整行；
