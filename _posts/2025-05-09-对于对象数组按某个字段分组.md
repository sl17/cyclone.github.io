## 对于对象数组按某个字段分组

#### 以 `business_module` 字段分组

```
const grouped = new Map()
for (const item of res.rows) {
	const key = item.business_module
	if (!grouped.has(key)) {
		grouped.set(key, [])
	}
	grouped.get(key).push(item)
}
grouped.forEach(item => {
	cardList.value.push(item)
})
```
