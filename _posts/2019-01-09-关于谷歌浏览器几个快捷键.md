## 关于谷歌浏览器几个快捷键

### 正常模式
- Ctrl + P：进入打印模式
- Ctrl + O：用浏览器打开文件
- Ctrl + U：查看网页源代码
- Ctrl + T：新建页面
- Shift + Ctrl + I: 打开浏览器Console界面
- Shift + Ctrl + N：打开浏览器无痕模式
- Shift + Ctrl + D：新建浏览器标签文件夹
- Shift + Ctrl + C：打开浏览器调试元素/样式界面
- Shift + Ctrl + T：恢复之前关闭的页面
- Shift + Ctrl + W：关闭当前窗口
- Shift + Ctrl + Q：关闭浏览器
- Shift + Ctrl + O：打开浏览器书签详情

### 调试模式下

- Shift + Ctrl + D：进入搜索模式
- Shift + Ctrl + M：进入手机模式
