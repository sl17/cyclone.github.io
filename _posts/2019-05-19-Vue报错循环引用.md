## Vue报错循环引用

### 错误重现

```
TypeError: Converting circular structure to JSON
    at JSON.stringify (<anonymous>)
```

>原因：在请求中传递的对象（我猜它是pagedoc）有一个循环引用，如：

```
var a = {};
a.b = a;
```

实际本地原因是因为：声明常量时 用了`model`
