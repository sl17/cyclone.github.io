## JS对数组行列转化



```
arr = [
    [1, 2, 3],
	[1, 2, 3],
	[1, 2, 3]
]
arr = arr[0].map((col, i) => {
	return arr.map(row => {
		return row[i];
	})
});
console.log(JSON.stringify(arr))
/*
	[
		[1,1,1],
		[2,2,2],
		[3,3,3]
	]
*/

```
