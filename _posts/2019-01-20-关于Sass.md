## 关于Sass

### 安装

```
npm install --save-dev sass-loader
//sass-loader依赖于node-sass
npm install --save-dev node-sass
```

### 配置

> 在build文件夹下的`webpack.base.conf.js`的`rules`里面添加配置

```
在：
module: {
  rules: [
    {
    	...
      loader: 'vue-loader'
      ...
    },
    {
      ...
      loader: 'babel-loader'
      ...
    },
    *在此添加*
  ]
}
添加如下代码：
{
  test: /\.sass$/,
  loaders: ['style', 'css', 'sass']
}
```
