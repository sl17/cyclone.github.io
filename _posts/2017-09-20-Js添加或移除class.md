## Js添加或移除class

#### 增加

```
document.getElementById("obj").classList.add("class1", "class2", "class3");

document.getElementById("obj").setAttribute("class", "class1" );
```

#### 删除

```
document.getElementById("obj").classList.remove("class1");
```
