## JS面向对象(OOP)之继承

```
function Person(name, age, sex){
	this.name = name;
	this.age = age;
	this.sex = sex;
	this.type = '人类';
	this.love = 'play';
}
```

通过apply实现继承

```
function Student(){
	Person.apply(this, arguments)
	this.love = '学习'
}
var zheng = new Student()
console.log(zheng.type)//人类
console.log(zheng)//Person{name: undefined, age: undefined, sex: undefined, type: "人类", love: "学习"}
```
